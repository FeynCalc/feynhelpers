## LToolsPath

`LToolsPath` is an option for `LToolsLoadLibrary`. It specifies the full path, to the LoopTools MathLink executable.

The default value is `FileNameJoin[{$FeynHelpersDirectory, "ExternalTools", "LoopTools", "LoopTools"}]`.
