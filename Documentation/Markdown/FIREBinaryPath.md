## FIREBinaryPath

`FIREBinaryPath` is an option for `FIRERunReduction`. It specifies the full path to the C++ FIRE binary.


The default value is `FileNameJoin[{$UserBaseDirectory, "Applications", "FIRE6", "bin", "FIRE6"}]`.
